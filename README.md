# OfficialPortal-Aiyouxi-Archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可能来自不同的来源或项目，以静态文件的形式集中存放，便于访问、查阅和共享。

## 项目简介

这是一个通用型的 HTML 页面归档仓库。仓库本身不针对特定域名或网站，而是作为多个独立页面的集合存储空间。每个页面通常是一个完整的、自包含的 HTML 文件，可能包含样式、脚本或其他前端资源。

通过本仓库，用户可以浏览或引用这些已归档的页面内容，而无需依赖原始服务器或特定网络环境。

## 目录说明

仓库根目录下直接存放 HTML 文件，或根据内容类型划分为子目录。常见的组织方式如下：

- `index.html`：仓库入口页面，提供归档文件列表或导航链接。
- `pages/`：存放主要归档的 HTML 页面文件。
- `assets/`：存放页面引用的公共资源文件（如 CSS、JavaScript、图片等）。
- `archive/`：存放历史版本或不再更新的页面文件。

目录结构可能随时间调整，请以仓库实际文件为准。

1. 直接访问仓库内的 HTML 文件，可通过 GitHub Pages 或原始文件链接查看。
2. 克隆仓库到本地，使用浏览器打开任意 HTML 文件。
3. 如需引用某个页面，请直接使用文件路径或 GitHub 原始文件 URL。
